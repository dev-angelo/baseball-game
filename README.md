# BaseballGame
BaseballGame Repository 는 CodeSquard 의 마스터즈 코스 온라인 테스를 위한 저장소입니다.  
step-1 Branch 는 야구게임 만들기 1단계에 해당하는 *step-1* 에 대한 요구사항을 구현 한 브랜치 입니다.  
## 각 Class 설명
### BaseballGameManager
### OfficialScorer
### PitchingResultGenerator
### ScoreBoard
### StatusPrinter
